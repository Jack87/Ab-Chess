Class AbChess

The AbChess class contructs an object to manage chess data as well as render a board.

Constructor Description
new AbChess(containerId[, options]) Create a new AbChess object.
Constant Description
DEFAULT_FEN The FEN string of the starting position in a classical chess game.

Constructor details

The AbChess class contructs an object to manage chess ...

new AbChess(containerId[, options])

Parameter Type Description
containerId String Required The id of the HTML container.
options Object Optional A configuration object. See options properties.
Options property Type Description Default value
clickable Boolean Optional If set to true, the chess pieces will be movable with a two-clicks method. true
draggable Boolean Optional If set to true, the chess pieces will be movable with a drag-and-drop method. true
flipped Boolean Optional If set to true, the chessboard will be oriented for black point of view. false
imagesExtension String Optional The extension of the chess pieces images. ".png"
imagesPath String Optional The path of the chess pieces images. "../images/wikipedia/"
legalMarksColor String Optional The CSS color of the legal moves indications. "steelblue"
markKingInCheck Boolean Optional If set to true, highlight the square of the king when it is attacked. true
markLastMove Boolean Optional If set to true, highlight the start and arrival squares of the last piece movement. true
markLegalSquares Boolean Optional If set to true, highlight the legal squares after a piece has been selected. true
markOverflownSquare Boolean Optional If set to true, highlight the overflown square during a drag and drop operation. true
markSelectedSquare Boolean Optional If set to true, highlight the square of a piece when it is selected. true
notationBorder Boolean Optional If set to true, the chessboard will have a notation border. true
width Number Optional The width of the chessboard in pixel. 360

Sample

Create an AbChess instance.

JavaScript var abChess; var abConfig = { clickable: true, draggable: false, flipped: false, showKingInCheck: true, showLegalMoves: false, width: 180 }; var containerId = "abContainer"; abChess = new AbChess(containerId, abConfig);
