    • Programmatically
  • Squares highlighting
    • King in check
    • Last move
    • Legal move possibilities
    • Overflown square
    • Selected piece
  • Test legal chess moves
  • Export a position
  • Import a position
  • Navigate in a game
  • Export a game
  • Import a game
