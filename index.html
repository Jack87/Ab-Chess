AB-Chess

A Javascript API for building chessboards and chessgames

Download AB-Chess

Do you want to embed a chessboard in your website ? Do you need to display chessgames ?
The Ab-Chess API provides you the tools to do it easily.

  • Embed an interactive chessboard
  • Get the legal moves
  • Play a game and store it
HTML <div id="chessboard"></div>
JavaScript var abChess; abChess = new AbChess("chessboard"); abChess.draw(); abChess.setFEN(abChess.DEFAULT_FEN);
