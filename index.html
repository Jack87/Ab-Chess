AbChess

All in one chess API for JavaScript

Download AbChess Currently version 0.3.0

Interactive chessboard

Do you want to have a chessboard in your website ?
Ab-Chess gives you the tools to embed an interactive chessboard. Easily configurable and customizable, it uses a CSS style sheet that can be modified.

Play chessgames

Do you need to play or replay chessgames ?
Ab-Chess provides you the methods to test the moves you want. It helps to validate FEN and PGN notations and it knows how a chessgame should work.

Light, easy and free

Ab-Chess is light and easy to use. It does not depend on any framework.
It is of course totally free. Just try and adopt it !
