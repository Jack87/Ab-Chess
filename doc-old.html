Class AbChess

The AbChess class creates an instance of an AbChess object. It provides an access to several methods. This class is mainly used to draw a chessboard and interact with it. The chess data are handled in a chessgame directly linked to the chessboard.

new AbChess(containerId[, options]) Create an AbChess instance.
Constant Description
DEFAULT_FEN The FEN string of the starting position in a classical chess game.

Constructor details

Create an instance of the AbChess class. Define the id of the HTML element to contain the desired chessboard. An options object can be also used to configure the instance.

new AbChess(containerId[, options])

Parameter Type Description
containerId String Required The id of the HTML container.
options Object Optional A configuration object. See options properties.
Options property Type Description Default value
animationSpeed String Optional The speed of the animated pieces. Allowed values are "slow", "normal", "fast" and "instant". "normal"
clickable Boolean Optional If set to true, the chess pieces will be movable with a two-clicks method. true
draggable Boolean Optional If set to true, the chess pieces will be movable with a drag-and-drop method. true
flipped Boolean Optional If set to true, the chessboard will be oriented for black point of view. false
imagesExtension String Optional The extension of the chess pieces images. ".png"
imagesPath String Optional The path of the chess pieces images. "images/wikipedia/"
legalMarksColor String Optional The CSS color of the legal moves indications. "steelblue"
markKingInCheck Boolean Optional If set to true, highlight the square of the king when it is attacked. true
markLastMove Boolean Optional If set to true, highlight the start and arrival squares of the last piece movement. true
markLegalSquares Boolean Optional If set to true, highlight the legal squares after a piece has been selected. true
markOverflownSquare Boolean Optional If set to true, highlight the overflown square during a drag and drop operation. true
markSelectedSquare Boolean Optional If set to true, highlight the square of a piece when it is selected. true
notationBorder Boolean Optional If set to true, the chessboard will have a notation border. true
width Number Optional The width of the chessboard in pixel. 360

Sample

Create an AbChess instance.

JavaScript var abChess; var abConfig = { clickable: true, draggable: false, flipped: false, showKingInCheck: true, showLegalMoves: false, width: 180 }; var containerId = "abContainer"; abChess = new AbChess(containerId, abConfig);

Methods details

draw()

Draw the chessboard in the HTML container element. This method should be called right after the constructor.

Return type : undefined

flip()

Change the orientation of the chessboard.

Return type : undefined

getActiveColor(index)

Return a character ("w" for white or "b" for black) indicating the active color in a position.

Return type : String

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

getFEN(index)

Return the FEN string of a position in the game.

Return type : String

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

getGameInfo(info)

Return an information stored in the game. Seven informations should be present in a game : Event, Site, Date, Round, White, Black, Result. Some other ones can be added.

Return type : String

Parameter Type Description
info String Required The name of the desired information.

getGameMoves()

Return an array of the moves stored in the game in a simple algebraic notation.

Return type : String[]

getGameMovesPGN()

Return an array of the moves stored in the game in PGN notation.

Return type : String[]

getLastPositionIndex()

Return the index of the last position of the game.

Return type : Number

getLegalMoves(index)

Return an array of the legal moves in a position.

Return type : String[]

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

getPGN()

Return the PGN string of the current game.

Return type : String

is50MovesDraw(index)

Return true if a position is a draw because of the 50 moves rule or false if it's not.

Return type : Boolean

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

isCheckmate(index)

Return true if a position is a checkmate or false if it's not.

Return type : Boolean

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

isInCheck(index)

Return true if the king of the active color is in check in a position or false if it's not.

Return type : Boolean

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

isInsufficientMaterialDraw(index)

Return true if the remaining material is insufficient to win a position or false if it's not.

Return type : Boolean

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

isLegal(index, move)

Return true if a move is legal in a position or false if it's not.

Return type : Boolean

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.
move String Required The move to test. It should be in the algebraic format.

isStalemate(index)

Return true if a position is a stalemate or false if it's not.

Return type : Boolean

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

isValidFEN(fen, onlyCheckRows)

Return true if a FEN string is valid or false if it's not.

Return type : Boolean

Parameter Type Description
fen String Required The Forsyth-Edwards Notation string to validate.
onlyCheckRows Boolean Required When set to true, the function will only check if the position rows are valid. When set to false, the function will validate the entire string.

isValidPGN(pgn)

Return true if a PGN string is valid or false if it's not.

Return type : Boolean

Parameter Type Description
pgn String Required The Portable Game Notation string to validate.

navigate(index)

Update the chessboard to a position in the game. The pieces are automatically locked until the last position is reached again.

Return type : undefined

Parameter Type Description
index Number Required The index of the desired position. It should be a positive integer.

onMovePlayed(callback)

Set a function to call each time after a move is played in the game.

Return type : undefined

Parameter Type Description
callback Function Required The function to call after a move has been played.

play(move[, promotion])

Play a move in the game and return the resulting FEN string.

Return type : String

Parameter Type Description
move String Required The move to play. It should be in the algebraic format.
promotion String Optional The promotion choice if needed. It will be ignored if the move is not a promotion.

reset()

Create a new game and set the chessboard to the initial starting position.

Return type : undefined

setFEN([fen])

Load a position on the chessboard from a FEN string. This does not affect the game in progress.

Return type : undefined

Parameter Type Description
fen String Optional The Forsyth-Edwards Notation to import. If omitted, the classical starting position will be set.

setGameInfo(info, value)

Set an information and its value in the game. Replace the information if it already exists. Create a new one if not.

Return type : undefined

Parameter Type Description
info String Required The name of the information to set.
value String Required The value of the information to set.

setPGN(pgn)

Load a chessgame from a PGN string. This will replace the game in progress.

Return type : undefined

Parameter Type Description
pgn String Required The Portable Game Notation to import.
