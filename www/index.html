AB-Chess

A Javascript library for building chessboards

Download AbChess-0.1

You want to embed a chessboard for you website ? You need to display chessgames ?

AbChess provides you the tools to do it easily.

  • Draw a chessboard
  • Make it interactive
  • Get the legal moves
  • Play a game and store it

HTML <div id="chessboard"></div>
Javascript var abChess; var options = { imagesPath: "../../images/wikipedia/" }; abChess = new AbChess("chessboard", options); abChess.draw(); abChess.setFEN(abChess.DEFAULT_FEN);
